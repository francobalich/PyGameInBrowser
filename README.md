
# PyGameInBrowser

En este proyecto se creara la base para ejecutar PyGame en en navegador con PygBag.

## Usa:
- Python 3.10.8

## Instalacion:
```bash
py -m pip install pygame
py -m pip install pygbag 
```
## Uso:
```bash
py -m pygbag .\PyGameDemo\
```
